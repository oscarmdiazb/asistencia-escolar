Importar lista (desde SIMAT)
¿Cómo importar la lista?
  1. Abre el listado de estudiantes del SIMAT en Excel.
  2. Crea o reorganiza las columnas en este orden exacto:
    SIMAT, Apellido1, Apellido2, Nombre1, Nombre2 (si un campo está vacío déjalo en blanco, pero mantén las comas)
  3. Guarda el archivo como CSV (separado por comas): Archivo → Guardar como → CSV UTF-8.
  4. Toca 📁 Importar archivo y selecciona el CSV guardado.
    O copia el contenido del archivo y pégalo en el cuadro de texto, luego toca 📥 Pegar y importar.
Ejemplo de línea válida:
1001234567, RODRÍGUEZ, VARGAS, ANA, MARÍA Los estudiantes ya existentes (mismo SIMAT) no se duplicarán.

Configuración inicial

Para conectar tu colegio necesitas una Google Sheet con el script de AulaPresente. Si aún no la tienes, pide el enlace de la plantilla al administrador del sistema.

Pasos:
1. Abre la plantilla de Google Sheet
2. File → "Hacer una copia" (queda en tu Drive)
3. Llena la pestaña Config (nombre y código del colegio)
4. Pega las listas de estudiantes en la pestaña Clases
5. Extensions → Apps Script → Deploy → Web app → Anyone
6. Copia la URL y pégala aquí abajo

📋 Gestionar listas de clase

Abre la hoja de Google Sheets para agregar o editar las listas de estudiantes directamente. Pega los datos desde Excel o SIMAT.

Formato de la pestaña "Clases":
Cada fila = un estudiante. Las columnas son:
classId · className · school · grade · aula · simat · apellido1 · apellido2 · nombre1 · nombre2
Todos los estudiantes de la misma clase deben tener el mismo classId.
